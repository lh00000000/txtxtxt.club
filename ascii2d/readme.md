a module for dealing with arrays like this for the sake of compositing layers of ascii art
```
[
  [{char: ' ', color: 'red'}, {char: '|', color: 'red'}, {char: ' ', color: 'red'}],
  [{char: '-', color: 'red'}, {char: '+', color: 'red'}, {char: '-', color: 'red'}],
  [{char: ' ', color: 'red'}, {char: '|', color: 'red'}, {char: ' ', color: 'red'}]
]
```
